DBA's framework for MySQL

download

The common_schema project is currently hosted by Google Code. Downloads are available at the Google Code common_schema project page

The common_schema distribution is a plain text SQL source file.

The common_schema distribution file supports MySQL 5.1, 5.5, Percona Server, with and without InnoDB Plugin. It should support MySQL 5.6, which is not yet GA at the time of this release of common_schema.

common_schema includes its own documentation (see help()). However, it is also possible to download the documentation as a bundled HTML archive.
