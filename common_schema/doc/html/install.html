install

common_schema distribution file is a SQL source file. To install it, you should execute it on your MySQL server.

There are many ways to import a SQL file into MySQL, listed below. Make sure to execute them as a privileged user, e.g. 'root', as installation involves creation of schema, tables, views & routines.

There are several common_schema distribution options, depending on your MySQL server version. Consult the download page

  • Within MySQL, issue: 
    mysql> SOURCE '/path/to/file.sql';
  • From shell, execute: 
    bash$ mysql < /path/to/file.sql
  • Use your favorite MySQL GUI editor, copy+paste file content, execute.

To verify installation, check that the common_schema database exists. e.g.:

root@mysql> SHOW DATABASES;
+--------------------+
| Database           |
+--------------------+
| information_schema |
| common_schema      |
| mysql              |
| sakila             |
| world              |
+--------------------+
