tokudb_views

SYNOPSIS

TokuDB views: views providing insight or operations on TokuDB tables

  • sql_alter_table_tokudb: Generate ALTER TABLE SQL statements for converting tables to TokuDB
  • tokudb_file_map: map TokuDB tables to files and common shell commands

DESCRIPTION

These views provide insight and operation on TokuDB tables. They come to overcome limitation in TokuDB (7.0.3 at this time of writing).
